# Theme Configuration Reference

This document maps configuration IDs from `options.php` to their functional impact in the theme.

| ID | Name | Description |
|---|---|---|
| `page_pjax` | PJAX Loading | Enables continuous music playback across page transitions. |
| `background_mode` | Background Type | Choose between Solid Color or Image. |
| `site_sa` | Sticky Sidebar | Enables sidebar widgets that follow scrolling. |
| `comment_ua` | Comment UA | Displays browser and OS info for commenters. |
| `createtime` | Site Launch Date | Used by the timer in the footer. |
| `social_bilibili` | Bilibili UID | Used to display Bilibili social links. |
| `post_cc` | Copyright Notice | Enables CC BY-SA 4.0 license display on posts. |
| `site_snow` | Site Snow | Enables a canvas-based snow animation. |
| `bibo_open` | Enable B-Blog | Enables Bilibili-style dynamic page integration. |
