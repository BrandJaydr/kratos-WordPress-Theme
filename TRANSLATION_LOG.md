# Translation Log

This document records the translation of the Kratos theme from Mandarin to English.

## Date: 2025-05-14

### Files Translated:
- `footer.php`: Translated site runtime and credits.
- `header.php`: Translated SEO metadata and technical comments.
- `index.php`: Translated Bilibili profile integration and sidebar labels.
- `page.php`: Translated license and sharing labels.
- `single.php`: Translated Bilibili profile integration.
- `comments.php`: Translated comment form labels and notes.
- `404.php`: Translated error message.
- `inc/core.php`: Translated menu labels.
- `inc/myfunction.php`: Translated word count, TOC labels, and email notification templates.
- `inc/widgets.php`: Translated widget titles, time-ago strings, and admin form labels.
- `inc/single-templates/single.php`: Translated post meta labels and navigation strings.
- `inc/content-templates/content.php`: Translated post meta and "Read More" labels.
- `inc/content-templates/content-status.php`: Translated status post meta labels.
- `static/js/kratos.js`: Translated layer messages, alerts, and site timer units.
- `static/js/pjax.js`: Translated console log messages.
- `options.php`: Translated the majority of theme options and descriptions.
- `inc/live2d/waifu-tips.json`: Fully translated the Live2D Waifu interaction messages.
- `inc/live2d/live2d.php`: Translated the other theme settings page labels and messages.

### Key Changes:
- Replaced Mandarin UI strings with English equivalents.
- Fixed mixed-language UI elements (e.g., "Following数" -> "Following").
- Ensured external links with anchor fragments (like Bilibili help pages) remain functional by not translating logic-critical fragments.
- Standardized terminology (e.g., "B站" -> "Bilibili", "个字" -> "words").
