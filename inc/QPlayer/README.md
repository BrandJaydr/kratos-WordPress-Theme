# QPlayer
A simple and small bottom floating music player plugin.

### Features
- Supports Netease Music parsing (Song, Album, Artist, Playlist).
- Supports cross-page playback (requires host to support curl).
- Customizable colors and CSS/JS.
- Random playback and auto-play options.

### Usage
1. Search and install the plugin in the background.
2. Go to Netease Music website to find the ID of the song/album/artist/playlist.
3. Configure the music list in the plugin settings.
4. Enable the plugin.
